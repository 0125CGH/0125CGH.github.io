Projects

Below are some of my current and past projects. Other ones I will be working on can be found at my GitHub.

  • dmr: For Battlehack Chicago, I decided to rethink one important aspect of the MapReduce distributed programming paradigm. Instead of sending mapper functions to cheap storage and processing nodes that are hard to maintain (including AWS's Elastic MapReduce in addition to traditional hardware), I thought that it might be cheaper and more scalable to send the mapper and the data to web devices. The logic is that almost all devices from phones to laptops browsing the web are using Javascript, so why not make them do some more work while browsing r/funny? Another incentive for this would be that website owners can make money on dmr jobs similar to ads. Meanwhile, those looking to do big data jobs can do so for cheaper and with less headache. A demo site with the given functionality and a dmr server is at the above link. As a side note, the name was originally Distributed MapReduce but was turned into the "dmr" acronym not only because it sounded better but also because it would be a tribute to Dennis Ritchie.
  • Model Viewer: This project was for a Computer Graphics course to demonstrate 3D rendering techniques and get experience with the full rendering pipeline. It is written in Javascript and WebGL, so it should display in any browser the supports WebGL.
  • MongoDB Scalability: For my Cloud Computing course, my partner and I deployed MongoDB and Node.js on Amazon EC2 instances and developed a simple URL API to submit MongoDB requests. The end goal was to test MongoDB performance by vertical (different EC2 instance types) and horizontal (sharding) scaling. In order to effectively test the effects of heavy load, we used the HTCondor parallel computing framework to simulate many transactions at once.
  • CareFree: For HackIllinois, my team developed an Android app that acts as a medical ID tag and also connects to hospitals and first responders for patients in need of care. The idea came from the fact that most people nowadays carry their phone everywhere. We decided to utilize this to devise an emergency response system that would allow first responders to know basic medical and personal information about the patient and be able to contact an emergency contact and administer aid accordingly. The submission won an award in Best Healthcare Hack.
  • Pokémon: A remake of Pokémon developed for Visual Studio and the X Window system written in C++ with the SDL graphics library. My partner and I learned quite a bit about the benefits of object oriented programming and basic sprite graphics in doing this project; we also discovered the wonders of version control.
  • This Website!: This website came out of a desire to learn CSS and Javascript (I had learned HTML earlier). It made sense to create a website while doing so and it's a work in progress so I will keep it updated. It's on GitHub as an public repo, if you want to check it out.
